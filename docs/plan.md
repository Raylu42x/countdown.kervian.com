# Finished Parts and Next Steps

## What's done

- ~43 holidays (religious, cultural, national, seasonal, fun)
- ~18 animated canvas effects
- Home grid sorted soonest first with category filters
- Card hover canvas previews (holiday's first effect, pre-warmed)
- Full-screen timer with box / big-seconds / donegreeting modes
- Responsive layout: wide-row / 2-up / single-column by aspect ratio
- Units drop off as the event approaches (no "0 days" box)
- Options panel: number size slider, animation speed slider, canvas picker with live thumbnails
- Custom countdowns: title, date/time, accent color (16 swatches), canvas, emoji builder (up to 5 emojis, fall/rise direction)
- Shareable hash links, all settings encode into the URL
- About page
- Selfrolling dates: fixed, nth-weekday, last-weekday, Easter (+offset for Mardi Gras), and published-table holidays auto-advance year over year
- Browser back button returns from timer to home
- Countdown numbers scale to fill large/wide viewports (no hardcoded 1080px cap)
- Emoji canvas listed first in the custom countdown canvas picker

## Possible next ideas

- Search box on the landing page
- "Add to calendar" link on the timer
- Sound/mute toggle that plays a chime at zero
- Pin favorites to the top of the list
- More canvas effects (menorah candles, dancing skeletons, etc.)
- Update lunar/observed holiday dates (Holi, Vesak, Jewish/Islamic) with newer published tables as they become available
